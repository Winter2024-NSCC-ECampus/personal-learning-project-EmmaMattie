# Task Management Program

## Overview
This is a Python program developed as part of a self-directed learning project. The program helps manage tasks through a simple menu system, demonstrating key Python programming concepts such as variables, loops, and functions.

## Features
- **Welcome Message**: Greets the user and introduces the program.
- **Menu System**: Displays options to add, view, or exit tasks.
- **Task Management**: Allows users to add tasks to a list and view the list of tasks.

## Usage
Upon running the program, the user is greeted with a welcome message and presented with a menu:
1. **Add a task**: Prompts the user to input a task, which is added to a task list.
2. **View tasks**: Displays all tasks currently stored in the list.
3. **Exit**: Ends the program.

The menu will continuously loop until the user selects **Exit**.
